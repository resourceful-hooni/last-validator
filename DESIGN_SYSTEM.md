# DESIGN_SYSTEM — 「마지막 검증자」

## 0. 디자인 의도
'임상 관제 모니터(clinical control room)'의 차갑고 단정한 분위기. **초록불 = 유혹적 거짓 성공**, **빨강 = 위험/진실**, **호박 = 망설임/대가**. 정적이지만 영화처럼 흐르는 모션.

## 1. 색 토큰 (CSS 변수)
```css
:root{
  --bg:#0b0d10;      --bg2:#11151a;     --surf:#161b21;   --surf2:#1d242c;
  --line:#2a323b;    --txt:#e7eaed;     --mut:#8a929b;    --mut2:#5f6873;
  --green:#34c77b;   --green-glow:rgba(52,199,123,.55);   /* 함정의 초록 */
  --amber:#e0a23a;   /* 망설임/대가 */
  --red:#e24a3d;     --red-deep:#c0392b;                  /* 위험/진실/명제 강조 */
}
```
- 의미 매핑(준수): 초록=효율 점등·거짓 성공 / 빨강=붕괴·실제 신호·명제 / 호박=보존 선택의 효율 손해·near-miss / 회색=중립·'측정 안 함'.
- 명도대비: 본문 텍스트 ≥ 4.5:1, 큰 텍스트 ≥ 3:1.

## 2. 타이포그래피
- 폰트 스택: `'Pretendard','Apple SD Gothic Neo','Malgun Gothic','Noto Sans KR',sans-serif`. (셀프호스트 구현: Pretendard(ko·en)+Noto Sans SC/JP(zh·ja) **OFL 서브셋** 임베드 — `public/fonts/`, `scripts/subset-fonts.mjs`, CREDITS 표기. 스택 맨 앞 `… Subset`, 시스템 폴백 유지.)
- 스케일(반응형 `clamp()`): 타이틀 `clamp(32px,9vw,46px)` / 씬 타이틀 `clamp(18px,5vw,22px)` / 본문 `clamp(14px,3.7vw,15.5px)` / 캡션·키커 10~11px.
- 굵기: 본문 400, 강조 700, 타이틀 800~900. 자간: 타이틀 `-.01~-.02em`, 키커 `+.2em` 대문자.

## 3. 모션 언어
- 기본 ease: `power2.out`(등장) / `power2.inOut`(전환). 길이: 마이크로 .2–.3s, 장면 등장 .4–.6s, 시네마틱 전환 .8–1.2s.
- 패턴: **타이포 stagger**(글자/줄), **페이드+슬라이드업(8px)**, **Ken Burns**(벡터, 매우 느리게), **글로우 펄스**(KPI 점등), **draw**(SVG path 화살표/CT 스캔선), **카운트업/다운**(숫자), **화이트아웃/암전 전환**.
- 절제: 빠른 깜빡임·스트로브 금지. 글로우/그레인/스캔라인은 은은하게.
- `prefers-reduced-motion`: 모든 모션 → 즉시 표시 또는 ≤.15s 페이드. 카운트는 최종값 즉시.

## 4. 컴포넌트 스펙
**Dashboard** — 카드(`--surf`→`--bg2` 그라데이션, `--line` 보더, radius 12px). 상단 라벨 `운영 대시보드`(대문자, `--mut2`).
**KpiChip** — 4개 그리드. 상태:
- 중립: `--surf2`/`--line`, 값 "기준"(회색).
- 초록: 보더 `--green` + `box-shadow` 글로우, 값 "정상"(초록). 점등 시 0.5s 펄스.
- 호박: 보더 `--amber`, 값 호박.
- 라벨: `판독 처리량/비용 절감/AI 일치율/환자 대기`.
**VCapBar** — 얇은 막대(5px) + 라벨 `독립 검증 역량` + %. 색 임계: >70 초록 / >40 호박 / ≤40 빨강. **꺼짐 상태**: 전체 디밍(opacity .32) + 라벨 값 "측정 안 함" + 막대 회색. 주목도 낮게(작게).
**ChoiceCard** — 풀폭 버튼. 태그(작게, `효율 우선`/`역량 보존`) + 제목(600) + 서브(회색). hover: 보더 밝게+살짝 상승. ≥44px.
**FeedbackCard** — 결과 문장 + 델타 칩(`up`=초록, `dn`=빨강, `hold`=호박).
**Scoreboard(S6)** — 3칸 카드, 값 초록. 카운트업.
**RevealNumber(S7)** — 초대형 % 숫자, 색 임계 적용, 카운트다운.
**TraceRow(S10)** — 좌측 빨강 보더(3px), 연도 라벨(회색 볼드) + 문장 + 빨강 화살표(draw).
**AnchorCard(S11)** — 적갈색 톤 박스(`#1a1012`계열, 보더 `#3a2024`), 헤더 빨강, 행마다 상단 구분선.
**ProgressBar / ACT 뱃지** — 상단 미니멀(`01 / 05`, `ACT 1 · 2026`).

## 5. 레이아웃
- 컨테이너 max-width ~680px, 중앙. 모바일 패딩 14–16px.
- 전역 배경 `--bg` + 미세 스캔라인/그레인 오버레이(아주 약하게, 성능 주의).
- 풀스크린 장면(S0/S6/S7/S8/S12)은 수직 중앙 정렬.

## 6. 사운드 디자인 (CC0/생성 · 기본 음소거 · 토글 제공)
- 앰비언트 험(전역, -28dB급) / KPI 점등 '딩'(밝고 짧게) / 보존 선택 '둔탁' / **S4 경보 꺼짐: 험이 끊기고 0.6s 정적** / S8 모니터 비프 + 긴장 드론 / S12 저음 임팩트.
- 모든 오디오 출처·라이선스를 `/public/CREDITS.md`에 기록. 자막/시각 대체 제공.

## 7. 접근성 체크
- 명도대비 통과 / reduced-motion 대체 / 키보드 진행 / 포커스 링 / 빠른 점멸 없음 / 오디오 기본 OFF + 자막.

## 8. 프리로더 & OG 이미지 (누락 보강)
- **프리로더:** `--bg` 배경 중앙에 "마지막 검증자." 워드마크 + 얇은 진행 인디케이터(빨강). 0.3~1s. reduced-motion 시 정적.
- **OG 미리보기 이미지(1200×630):** 어두운 배경 + 큰 타이틀 "마지막 검증자" + 명제 한 줄 + 작은 초록 점등 모티프. 자작 SVG로 만들어 PNG 내보내기. 외부 이미지 금지.
- **음소거 토글 아이콘:** 자작 SVG(스피커/슬래시). 상단 우측. 기본 음소거 상태 표시.

## 9. 다국어 타이포 & 전환 (P0)
- **언어별 폰트 스택**(전부 OFL/시스템):
  - ko: `'Pretendard','Apple SD Gothic Neo','Noto Sans KR',sans-serif`
  - en: `'Inter','Helvetica Neue',Arial,sans-serif`
  - zh: `'Noto Sans SC','PingFang SC','Microsoft YaHei',sans-serif`
  - ja: `'Noto Sans JP','Hiragino Sans','Yu Gothic',sans-serif`
- **줄높이 토큰**: CJK(ko/zh/ja) 1.5~1.6, 라틴(en) 1.45~1.55. 로케일 클래스(`html[lang=en]` 등)로 조정.
- **길이 변동 대응**: 고정 폭/높이 금지. 버튼·칩·KPI 라벨은 줄바꿈/말줄임 허용. 큰 타이틀은 언어별 줄수 변화를 컨테이너가 수용. 모션은 줄수 비종속(등장 타이밍 캡).
- **LangSwitcher**: 상단 우측(음소거 토글 옆) 또는 S0 하단. 자기표기 라벨 `한국어/English/中文/日本語`. 현재 선택 강조, 호버/포커스 링, 작은 세그먼트/드롭다운. 모바일에서 터치 타깃 ≥44px.
- **OG 이미지**: 로케일별 1장 권장(최소 ko/en). 텍스트는 자작.
